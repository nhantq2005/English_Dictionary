# English Dictionary App

A modern, feature-rich English Dictionary Android application designed to help users look up word definitions, manage vocabulary, and enhance their English learning experience.

## Features

- **Word Search:** Quickly search for English word definitions and pronunciations.
- **Flashcards:** Create and review flashcards to memorize vocabulary efficiently.
- **Save Words:** Bookmark and organize words for future reference.
- **User-Friendly Interface:** Clean, intuitive, and responsive UI for seamless navigation.
- **Offline Support:** Access saved words and flashcards without an internet connection.

## Getting Started

### Prerequisites
- Android Studio (latest version recommended)
- Android SDK
- Gradle

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Open the project in Android Studio.
3. Build and run the app on an emulator or physical device.

## Project Structure

- `app/src/main/java/` - Application source code
- `app/src/main/res/` - Resources (layouts, drawables, etc.)
- `app/src/main/AndroidManifest.xml` - App manifest
- `build.gradle.kts` - Project build configuration
